# agentic_data_access

This repository includes resources on the following article. 

Monica Marconi Sciarroni, Emanuele Storti. "Agent-based Service Orchestration for Context-Aware Data Access in Industrial IoT", to appear in the Proc. of the 2nd Workshop on Large Language Models for Service-Oriented Architectures and Systems Design
in conjunction with the 38th International Conference on Advanced Information Systems Engineering (CAISE 2026), Verona, Italy.

In the experiments folder, each csv file includes the Entity Recognition evaluation for the two models Gemini 2.5 Flash-Lite and LLama 3.3, with a different number of sensors (5,25,50).
